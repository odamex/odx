Server Hosting Coming Soon

The server hosting feature will provide a streamlined interface for hosting your own Odamex game servers directly from ODX.

This feature is currently under development and will be available in a future update.

Easy Configuration

Configure server settings through an intuitive interface - no manual config file editing required.

Quick Start

Launch a server with sensible defaults in seconds, or customize every detail for advanced hosting.

Server Management

Start, stop, restart, and monitor your servers. View real-time player counts and server status.

Multiple Servers

Host multiple game servers with different configurations. Perfect for running various game modes.

WAD Management

Easily select IWADs and PWADs from your library. Support for custom map rotations and mod configurations.

Server Analytics

Track player activity, match history, and performance metrics for your hosted servers.
